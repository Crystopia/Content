<TagLinks tag="howtojoin" title="Wie tritt man Crystopia bei?" type="text-5xl" />
<br /><br />
<TagLinks tag="step1" title="Schritt 1: Starte Minecraft" type="text-4xl" />
<br />
> Öffne den Minecraft-Launcher auf deinem Computer und melde dich mit deinem Minecraft-Konto an.

![Öffne den Minecraft-Launcher](https://cdn.xyzhub.link/u/vSBov3.png)

1. Klicke auf „Spielen“, um Minecraft zu starten  
2. Wähle die neueste Minecraft-Version (1.21.4)
<br /><br />
<TagLinks tag="step2" title="Schritt 2: Gehe zu Multiplayer" type="text-4xl" />
<br />
1. Klicke im Hauptmenü von Minecraft auf „Mehrspieler“.  
2. Wähle „Server hinzufügen“, um einen neuen Server hinzuzufügen

![Klicke auf Mehrspieler](https://cdn.xyzhub.link/u/eyp53G.png)  
![Neuen Server hinzufügen](https://cdn.xyzhub.link/u/DSFH7U.png)
<br /><br />
<TagLinks tag="step3" title="Schritt 3: Füge den Server Crystopia.net hinzu" type="text-4xl" />
<br />
1. Gib im Feld „Servername“ einen Namen für den Server ein, z. B. „Crystopia“.  
2. Gib im Feld „Serveradresse“ die Adresse des Servers ein: crystopia.net  
3. Klicke auf „Fertig“, um den Server zu speichern.

![Füge den Server Crystopia.net hinzu](https://cdn.xyzhub.link/u/YFwnw2.png)
<br /><br />
<TagLinks tag="step4" title="Schritt 4: Tritt dem Crystopia-Server bei" type="text-5xl" />
<br />
1. Du wirst zurück zur Liste der Mehrspieler-Server geleitet.  
2. Wähle den Crystopia-Server aus der Liste und klicke auf „Server beitreten“.

![Tritt Crystopia.net bei](https://cdn.xyzhub.link/u/zNCOBY.png)
<br /><br />
<TagLinks tag="step5" title="Schritt 5: Viel Spaß auf Crystopia!" type="text-5xl" />
<br />
> Sobald du dem Server beigetreten bist, befindest du dich in der Welt von Crystopia. Erkunde die Welt, baue und kreiere, interagiere mit anderen Spielern und genieße das Spiel!
